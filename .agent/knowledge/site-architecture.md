# Site Architecture & Key Pages

The Palazzo Antico website is a high-performance static site designed for luxury hospitality conversion.

## Key Pages

- `index.html`: Hero introduction with "breathing" animation and overview.
- `rooms.html`: Catalog of rooms using the Bento Grid display system.
- `chef-rocco.html`: Culinary excellence and personalized services section.
- `retreats.html`: Experience-driven landing page for art, embroidery, and wellness retreats.
- `about.html`: Family history (Anastasiadis family) and restoration story.

## Technical Stack

- **Framework**: Vanilla HTML5, CSS3, and JavaScript.
- **SEO**: Structured Data (JSON-LD) implemented for Organization, Hotel, and Local Business.
- **Animations**: Custom CSS transforms and keyframes for smooth, premium transitions.
- **Performance**: Targeting a high Lighthouse score via WebP image optimization and lazy loading.
